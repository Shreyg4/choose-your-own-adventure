Split-Pane Authoring UI — text editor on the left, live graph on the right.
Handling Non-Story Pages — filtering out metadata pages like "About the Author" (page 117) and book lists (page 119).
Export & Publishing Pipeline — static HTML generation plus re-running the canonical Python scripts.
